Compatível com a maioria das distribuições Linux.

Para que a AppImages Shop funcione corretamente, você precisa instalar algumas dependências. Consulte a seção Dependências para mais detalhes.

Instalação de Dependências

A AppImages Shop requer **PyGObject** (para a interface GTK+) e a biblioteca **Requests** para buscar o catálogo de aplicativos. Utilize os comandos abaixo para sua distribuição.

Debian/Ubuntu/Mint

Abra um terminal e execute:

sudo apt update
sudo apt install python3-gi python3-requests gir1.2-gtk-3.0

Fedora/CentOS/RHEL

Abra um terminal e execute:

sudo dnf install python3-gobject python3-requests gtk3

Arch Linux

Abra um terminal e execute:

sudo pacman -S python-gobject python-requests gtk3

openSUSE

Abra um terminal e execute:

sudo zypper install python3-gobject python3-requests gtk3
